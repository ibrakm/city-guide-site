## LESSON PLAN TEMPLATE

Each lesson (1-30) follows this exact structure:

### 1. Leading Discussion (7-8 minutes)
- Teacher opening script
- 3-4 warm-up questions
- Brainstorming activity  
- Critical thinking question
- **Full teacher guidance for weaker students**

### 2. Vocabulary & Reading (10-12 minutes)
- Complete vocabulary lists (15-25 words)
- Pronunciation guides
- Example sentences
- Reading text (6-10 sentences)
- Comprehension questions
- Discussion questions
- **Pair-work reading activity**

### 3. Language Focus (15 minutes)
- Key grammar/expression patterns
- 4-5 example sentences
- Scripted dialogue (6-10 lines)
- Step-by-step teaching instructions
- Repetition drills
- **Pair/group role-play**

### 4. Guided Practice (10 minutes)
- Gap-fill exercises (5-6 items)
- Sentence completion
- Transformation exercises
- Oral practice questions
- **Pair-work checking activity**
- Full answer keys provided

### 5. Communicative Task (10 minutes)
- Real-life scenario role-play
- Information gap activities
- Surveys, interviews, or projects
- Follow-up prompts for teacher
- **Student-centered, maximum speaking time**

### 6. Wrap-up & Mini-test (5 minutes)
- 3-5 assessment questions
- Quick written check
- Reflection question
- **Peer testing option**

### 7. Homework (Optional)
- Writing task (80-100 words)
- OR Recording task (spoken practice)
- Clear examples provided

### 8. High Achievers Section
- 4-5 extension challenges
- Advanced vocabulary
- Creative tasks
- Research projects
- Higher-order thinking activities

## HOW TO CONTINUE CREATING REMAINING LESSONS

To create Lessons 7-30, follow the exact template and structure of Lessons 1-6:

### For Each Lesson:

1. **Identify the topic** (e.g., "Introducing Yourself")

2. **Create Leading Discussion:**
   - Opening script connecting to students' lives
   - 3-4 warm-up questions (personal, relevant)
   - Brainstorming (list vocabulary/ideas on board)
   - Critical thinking question (why/how/what if)

3. **Build Complete Vocabulary List:**
   - 15-25 key words/phrases
   - Pronunciation guides
   - Example sentences for each
   - Cultural examples when possible

4. **Write Reading Text:**
   - 6-10 sentences (100-120 words)
   - Use target vocabulary
   - First-person narrative often works well
   - Moroccan student character
   - 5 comprehension questions
   - 2-3 discussion questions

5. **Design Language Focus:**
   - 1-2 key grammar patterns (simple, clear)
   - 4-5 example sentences
   - Scripted dialogue (6-10 lines)
   - Repetition and role-play instructions

6. **Create Guided Practice:**
   - Gap-fill (5-6 items)
   - Sentence completion/transformation
   - Oral drill questions
   - Answer key

7. **Plan Communicative Task:**
   - Real-life scenario (10 minutes)
   - Pair or small group
   - Clear instructions
   - Follow-up questions
   - Reporting to class

8. **Design Mini-test:**
   - 3-5 quick questions
   - Mix oral and written
   - Reflection question

9. **Write Homework:**
   - Writing (80-100 words)
   - OR Recording (1-2 minutes)
   - Clear example

10. **Add High Achiever Challenges:**
    - 4-5 extension tasks
    - More complex language
    - Creative/research elements
